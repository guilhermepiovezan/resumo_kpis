Resumo KPIs - Automação Google Sheets

Script para consolidar KPIs em uma aba única no Google Sheets usando Google Apps Script

Sobre o Projeto

Este projeto automatiza a consolidação e resumo de KPIs mensais, facilitando a análise dos dados nacionais e regionais em uma planilha Google Sheets.

Como usar

  1. Abra sua planilha no Google Sheets.
  2. Vá em Extensões > Apps Script.
  3. Copie o código do script resumk-kpis.gs e cole no editor.
  4. Salve e execute a função resumoBasicoERegional2025().
  5. Confira os resultados na aba Resumo_KPIs.
